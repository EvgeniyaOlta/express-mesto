# Проект Mesto фронтенд + бэкенд

Nodejs
Express
MongoDB

## Точка входа

Файл app.js.

## Директории

`/routes` — файлы роутера  
`/controllers` — контроллеры
`/errors` — компоненты ошибок
`/models` — схемы card и user


## Запуск проекта

`npm run start` — запускает сервер   
`npm run dev` — запускает сервер с hot-reload
